Overview

What is it ?

BookNaviger is a software which allow you to :

  • organize your books folder by serie and albums
  • read theses books

Main features

  • Organize your books by series' name
  • Organize each books album within its serie
  • Preview the selected album
  • Read the selected album
    • Rotate the picture
    • Dual page's read mode
    • Zoom in or out the picture, or fit to the horizontal or vertical size
  • Multi-profiles (one profile is one books' root folder)
  • Generate a web page with the list of your books (either a simple list - just text - , or a detailed one with thumbnails of each album)

Compatibility & requirements

Cross-Plateforms : BookNaviger is written in Java. It means it works (at least, it's supposed to work) everywhere that the Java Virtual Machine (JVM) is installed. The JVM version needed is Java 7 (you can get it here).

Known bugs

Some bugs still exists on Mac OS X :

  • Folders with foreign charaters (à, é, è, ë, ê, ù, ...) can't be read if the file is launched from the interface. You need to launch it from the terminal with the command java -jar ./BookNaviger.jar.
  • The trackpad special mouvements (swipe, rotate, magnify) can't control the navigation anymore. Boths theses bugs are related to Java 7 for Mac OS X which no longer relies on Apple. Nothing i can do on my side. Wait and see...
